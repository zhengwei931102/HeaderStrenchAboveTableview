# HeaderStrenchAboveTableview
## 做成类似tableheaderview，滑动到顶部悬停，segment切换tableview，且带下拉刷新上拉加载<br>
### 借鉴JXCategoryView实现自定义segment，借鉴JXPagingView实现滑动到顶部悬停效果， 且segment切换tableview借鉴MJRefresh实现下拉刷新上拉加载～
